BookVerse is a book discovery and review platform designed for passionate readers who love to share their literary experiences. Built with clean HTML and elegant CSS, our platform offers a sophisticated space for book enthusiasts to explore, review, and discuss their favorite reads.

Our mission is to connect readers around the world through the shared love of literature, providing an intuitive and visually stunning interface that makes discovering your next great read effortless.

Full-Stack Web Development Course

Academic Project

This website is the culmination of Units 13-20 from the Full-Stack Web Development course at Lev Academic Center, taught by Prof. Shlomo Kipnis.

Technologies Used

Pure HTML & CSS • Responsive Web Design • CSS Grid & FlexBox • CSS Animations • Modern Web Standards

Project Goals

Create a fully responsive, content-rich static website demonstrating mastery of front-end fundamentals including forms, tables, media integration, and responsive layouts.
